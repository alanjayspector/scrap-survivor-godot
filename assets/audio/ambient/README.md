# Ambient Audio Files

**Required**: 3 ambient sound files

See `../AUDIO_SOURCING_GUIDE.md` for sourcing instructions.

## File List:

1. `wave_start.wav` - Dramatic sting/alarm (2-3 seconds)
2. `wave_complete.wav` - Victory fanfare/power-up (3-4 seconds)
3. `low_hp_warning.wav` - Heartbeat or alarm beep (1-2 seconds, loopable)

**Source**: Kenney Audio Packs (Digital Audio, Sci-Fi Sounds, UI Audio)
**License**: CC0 (Public Domain)
**Target Size**: < 500 KB each, < 1.5 MB total

**Special Notes**:
- `low_hp_warning.wav` should be set to LOOP in Godot import settings
- `wave_start.wav` and `wave_complete.wav` should be ONE-SHOT (no loop)
