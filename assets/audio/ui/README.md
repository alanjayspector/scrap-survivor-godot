# UI Audio Files

**Required**: 3 UI sound files

See `../AUDIO_SOURCING_GUIDE.md` for sourcing instructions.

## File List:

1. `button_click.wav` - Soft beep or tap (short, subtle)
2. `character_select.wav` - Confirm chime/power-up (positive, clear)
3. `error.wav` - Negative buzz/denied (short, clear denial)

**Source**: Kenney Audio Packs (UI Audio, Digital Audio)
**License**: CC0 (Public Domain)
**Target Size**: < 50 KB each, < 150 KB total

**Special Notes**:
- All UI sounds should be SHORT (< 0.5 seconds)
- `button_click.wav` should be VERY subtle (will play frequently)
- `error.wav` should be clearly distinguishable from success sounds
